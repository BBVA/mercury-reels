\doxysubsection{Detailed Description}
A container class to hold clips (sequences of events). 

To simplify the Python interface, the object has a \mbox{\hyperlink{classreels_1_1TimeUtil_ae45cac3f4e552d0128d3b82039896456}{set\+\_\+time\+\_\+format()}} method.
