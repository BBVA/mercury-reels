\doxysubsection{Detailed Description}
An generic block structure to store object state in a Python-\/friendly way. 

This structure is 64encoded to 8K (3 input bytes == 24 bit -\/\texorpdfstring{$>$}{>} 4 output chars == 24 bit). Therefore, its size is 6K.
